Estruturas condicionais

Condição simples

Estruturas condicionais servem para alterar o fluxo de execução do meu algoritmo. Após um teste de condição, se a condição for verdadeira, o fluxo de execução do codigo interno à estrutural condicional executa. Caso o contrario, codigo salta para o final da estrutura condicional

if(teste de condição){ comandos executados quando a condição é verdadeira }

Condicional composta

if(teste de condição){ comandos executados quando a condicao é verdadeira } else { comandos executados quando a condição é falsa }
